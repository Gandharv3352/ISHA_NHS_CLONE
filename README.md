## ğŸ“Œ About The Project

**ISHA NHS Clone** is a frontend-focused website project inspired by the official NHS website.  
It replicates the design style, layout, and navigation to demonstrate skills in **HTML, CSS, JavaScript, and PHP**, along with real-world deployment using **Docker and Apache**.

This project is intended for **learning, practice, and portfolio demonstration purposes only**.
